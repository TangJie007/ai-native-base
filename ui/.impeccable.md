# Impeccable Design Context — 灵境 (LingScape)

## Target audience
Windows 桌面美化爱好者、AI 工具尝鲜者、创意工作者，18-35 岁，对桌面个性化有强烈需求。他们在使用 Wallpaper Engine 或 Lively Wallpaper 后想要更多 AI 驱动的独特体验。

## Use cases
1. 用一句话描述想要的桌面效果，AI 生成独一无二的壁纸
2. 上传参考图，AI 分析风格后生成类似但不同的壁纸
3. 浏览、管理、应用已生成的壁纸库
4. 配置和管理 AI 模型 API Key
5. 首次使用时，通过向导快速完成 API 配置并生成第一张壁纸

## Brand personality/tone
**"灵境流光"** — 空灵、深邃、流动、有温度。界面如同一扇通往活着的世界的窗户。不是冰冷的工具，而是有灵性的创作伙伴。

- **空灵**: 暗色基调，极光般的渐变流光，呼吸感的动效
- **深邃**: 多层纵深，墨韵般的背景质感，不是扁平的黑暗
- **流动**: 过渡动画流畅，状态切换有水波感，不是生硬跳转
- **温度**: 暖金色点缀，人性化的文案，不是冷冰冰的技术界面

## Aesthetic direction
**克制的大胆** — 在深邃的暗色中，用精确的光来表达。不是全息霓虹、不是毛玻璃泛滥、不是紫色渐变。灵感来自：极光、水墨、深海生物荧光、中国传统山水画的留白。

## Differentiation
Aurora gradient border on key interactive elements — 不是静态边框，而是流动的极光色彩，随状态变化而改变。这是灵境的视觉签名。
